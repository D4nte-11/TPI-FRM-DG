Gestión de Datos de Países en Python

 Descripción del proyecto

Este proyecto consiste en el desarrollo de una aplicación de consola en Python destinada a la gestión de información sobre países. El sistema permite cargar datos desde un archivo CSV, almacenarlos mediante estructuras de datos como diccionarios, y realizar diferentes operaciones de consulta y administración.

La aplicación cuenta con funcionalidades para agregar y actualizar países, realizar búsquedas por nombre con coincidencias parciales o exactas, aplicar filtros por continente, población y superficie, ordenar los datos según distintos criterios y generar estadísticas básicas sobre la información almacenada.

El objetivo principal del proyecto es aplicar conceptos fundamentales de Programación 1, incluyendo manejo de archivos CSV, estructuras de datos, funciones, validaciones, ordenamientos y procesamiento de información mediante Python.

El sistema fue desarrollado siguiendo una estructura modular, separando las responsabilidades mediante funciones independientes para mejorar la organización, legibilidad y mantenimiento del código.
